# calendar-app

Multi-tenant Calendly-style booking tool for trade contractors.
A visitor lands on a contractor's website, picks an available time slot in a branded widget, and walks away with a confirmed booking in their inbox — no phone tag, no back-and-forth.

**Stack:** Next.js 16 (App Router) · Supabase (Postgres + Auth + Storage) · Tailwind CSS v4

## Getting started

Full setup instructions land in Plan 03. For now:

```bash
npm install
npm run dev
```

Copy `.env.example` to `.env.local` and fill in the three Supabase values before running the app against real data.
