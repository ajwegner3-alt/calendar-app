<objective>
Modify `POST /api/bookings` to support N-per-slot capacity. Read `max_bookings_per_slot` from the resolved event type. Retry the booking INSERT across `slot_index = 1..N` on Postgres 23505. After exhausting all N values, return 409 with code `SLOT_TAKEN` (capacity=1) or `SLOT_CAPACITY_REACHED` (capacity>1) per CAP-07.

Purpose: The atomic Postgres-side guarantee from Plan 03 protects against double-booking. This plan wires the application logic to use it correctly with the right error semantics for the booker UI.

Output: `app/api/bookings/route.ts` updated with capacity-aware retry loop. Existing test `tests/bookings-api.test.ts` continues to pass (regression check). Existing test `tests/race-guard.test.ts` may need a small update if it asserts on the exact error code string.
</objective>

Per RESEARCH.md §Pattern 1 example block (verbatim):

```typescript
// Replace the existing single INSERT block (currently around lines 182-222 per RESEARCH.md note).
// The retry loop attempts slot_index=1, then 2, then 3, ..., up to maxBookingsPerSlot.
// On 23505, the slot+slot_index combination is taken — try the next slot_index.
// On any other error, fail fast.

let booking: { id: string; start_at: string; end_at: string; booker_name: string; booker_email: string; booker_phone: string | null; booker_timezone: string; answers: unknown } | null = null;
let insertError: { code?: string; message: string } | null = null;

for (let slotIndex = 1; slotIndex <= maxBookingsPerSlot; slotIndex++) {
  const result = await supabase
    .from("bookings")
    .insert({
      // ...all the existing fields the route currently inserts...
      slot_index: slotIndex,
      status: "confirmed",
    })
    .select("id, start_at, end_at, booker_name, booker_email, booker_phone, booker_timezone, answers")
    .single();

  if (!result.error) {
    booking = result.data;
    insertError = null;
    break;
  }

  insertError = result.error;

  if (result.error.code !== "23505") {
    // Non-capacity error: do not retry. Propagate.
    break;
  }
  // 23505 = that (event_type_id, start_at, slot_index) is already taken. Try next slot_index.
}

if (!booking) {
  if (insertError?.code === "23505") {
    // All slot_index values 1..maxBookingsPerSlot were taken — capacity fully reached.
    const code = maxBookingsPerSlot === 1 ? "SLOT_TAKEN" : "SLOT_CAPACITY_REACHED";
    const message =
      maxBookingsPerSlot === 1
        ? "That time was just taken. Please choose a different time."
        : "That time is fully booked. Please choose a different time.";
    return NextResponse.json(
      { error: message, code },
      { status: 409, headers: NO_STORE },
    );
  }
  // Non-capacity error
  console.error("[bookings] INSERT failed:", insertError);
  return NextResponse.json(
    { error: "Booking failed. Please try again.", code: "INTERNAL" },
    { status: 500, headers: NO_STORE },
  );
}
```
